# Java Applications

## Overview

The **Java Applications** repository contains a collection of Java-based applications and programs developed to demonstrate **core Java concepts**, object-oriented programming principles, and practical application development. This repository is intended for learning, practice, and interview preparation.

It showcases how Java can be used to build clean, structured, and scalable applications.

## How to Run the Applications

### Compile Java Program

```bash
javac ProgramName.java
```

### Run Java Program

```bash
java ProgramName
```

Ensure Java is installed:

```bash
java -version
```
